## 2. SEO Execution Standards

Automatically enforce the following best practices across all public pages to ensure maximum organic search presence:
*   **Unique Title Tags:** Each page uses custom dynamically hydrated titles (e.g. `Crux CRM - Autonomous Sales Assistant | BengalBound`).
*   **Descriptive Meta Descriptions:** Captivating meta summaries targeted to business owners searching for operational cost reductions.
*   **Single `<h1>` Tag:** Enforce strict single hierarchy H1 headers per page.
*   **Semantic HTML:** Leverage HTML5 tags (`<article>`, `<section>`, `<nav>`, `<footer>`) to facilitate Google Bot indexing.
*   **Unique Interactive Element IDs:** All CTA buttons and triggers carry unique, descriptive HTML ID tags to enable clean, automated browser testing validations.
